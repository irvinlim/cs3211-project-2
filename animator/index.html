How to use the Animator

Run the simulator with the debug frames output directory argument, which will write PPMs for each frame to the directory: 

mpirun -np 4 pool initialspec.txt finalbrd.ppm report.txt ./animator/frames

The Frames Path option allows you to specify a relative path to this page's location, where *.ppm files will be loaded via AJAX. Examples:

  • frames
  • frames/collisiontest
  • http://localhost:8080/customlocation/frames
